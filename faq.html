💬General
What is AI citation tracking and why does it matter?
+
AI citation tracking is the practice of monitoring how often your brand, website, or product is mentioned in the responses that AI chatbots like ChatGPT, Claude, Gemini, and Perplexity generate. As more users turn to AI interfaces instead of traditional search engines to find products and recommendations, appearing in those AI-generated answers has become a new form of brand visibility — sometimes called "Generative Engine Optimization" (GEO).
Which AI platforms does the extension support?
+
Currently four platforms: ChatGPT (OpenAI), Claude (Anthropic), Gemini (Google), and Perplexity. You can use any combination — even just one if that's all you need. Gemini is free; the others require paid API keys.
How accurate is the citation detection?
+
Very accurate. v2.0 introduced smart fuzzy-matching that detects your brand across multiple formats — domain ("loom.com"), name ("Loom"), and possessives ("Loom's"). This produces 3–5× more citations than simple exact-match approaches. It's not perfect — sometimes AI models use indirect references — but it covers the vast majority of meaningful mentions.
Do I need to create an account or log in?
+
No account needed — ever. Install the extension, add your API key(s) in the Settings tab, and start checking. All your data stays locally in your browser. We have no user accounts, no database, no tracking.
⚙️Setup & Installation
How do I install the Chrome extension?
+
Download and unzip the extension file. In Chrome, go to chrome://extensions, enable Developer Mode in the top right, click "Load unpacked", and select the unzipped folder. The extension will appear in your toolbar immediately. Full guide: How It Works →
How do I get a free Gemini API key?
+
Visit aistudio.google.com/app/apikey, sign in with your Google account, and click "Create API Key". Copy the key (it starts with "AIza...") and paste it into the extension's Settings tab under "Gemini API Key". No credit card required. The whole process takes under 2 minutes.
Does it work on browsers other than Chrome?
+
Currently Chrome only (and Chromium-based browsers like Edge and Brave). Firefox and Safari support is on the roadmap but not yet available.
Can I use multiple platforms simultaneously?
+
Yes — just add all your API keys in Settings. When you run a check, it queries all platforms that have a key configured in parallel, and shows results for each platform separately in the Results tab.
💰Pricing & API Keys
Is the Chrome extension free?
+
The extension itself is permanently free to download and use. The only costs are the AI API calls you make directly to providers. Gemini's free tier means you can run an unlimited number of checks at $0. See Pricing → for a full breakdown.
What does one "check" cost?
+
One check = one query sent to one platform. Gemini: $0. Perplexity: ~$0.005. Claude: ~$0.015. ChatGPT: ~$0.030. So if you run 5 queries across all 4 platforms, that's roughly $0.25 total (with Gemini contributing $0).
Are there any hidden fees or subscriptions?
+
None whatsoever. We charge nothing. The extension is free. API costs go directly to the AI providers (Google, Anthropic, OpenAI, Perplexity) through your own accounts — we're not involved in that transaction at all.
⚡Features
How does the competitor comparison feature work?
+
Enter up to 5 competitor domains alongside your own in the Compare tab. The extension runs the same queries for every domain across all configured AI platforms, then produces a side-by-side ranking table with citation counts, gap scores, and AI-generated recommendations on which queries to focus on.
What does "Auto-Generate Queries" do?
+
It uses the AI to infer your industry and product category from your domain name, then generates 5–10 high-signal search queries that are most likely to surface citations. You can edit, remove, or add to the generated list before running a check. It works for both your domain and competitor domains.
Can I export or download results?
+
Yes. From the Results tab, you can copy all results to your clipboard in a formatted layout, or download a CSV file containing each query, platform, citation count, and whether citations were found. The CSV is ideal for monthly reporting or tracking trends in a spreadsheet.
How many competitor domains can I compare at once?
+
Up to 5 competitor domains at a time, plus your own domain — so 6 total in a single comparison run. This covers most use cases without generating excessive API usage or wait times.
🔒Privacy & Security
Is my API key stored securely?
+
Yes. API keys are stored using Chrome's built-in secure local storage (chrome.storage.local), which is sandboxed to the extension. They are never sent to our servers — because we don't have any. Queries go directly from your browser to the AI provider's API.
Do you collect any data or track usage?
+
No. We collect nothing. There are no analytics, no telemetry, no usage tracking of any kind. The extension has no backend server. Everything runs locally in your browser.
What permissions does the extension request?
+
Only "storage" (to save your API keys and settings locally) and host permissions for the four AI provider APIs (anthropic.com, openai.com, googleapis.com, perplexity.ai). It doesn't request access to your tabs, browsing history, clipboard, or any other sensitive browser data.
🛠️Troubleshooting
I'm getting a 401 error — what does it mean?
+
A 401 error means your API key is invalid or has been revoked. Go to the relevant API provider's dashboard, create a new key, and update it in the extension's Settings tab. Make sure there are no extra spaces when pasting the key.
I'm getting 0 citations — is something wrong?
+
Zero citations can be a completely normal result, especially for newer or lesser-known brands. Test first with a well-known domain like "salesforce.com" or "notion.so" to verify the extension is working. If those return citations but yours returns zero, it means AI models aren't currently referencing your brand — which is exactly what you need to know so you can fix it.
The extension popup is blank or not loading.
+
Try these steps in order: (1) Reload the extension at chrome://extensions using the refresh icon. (2) Hard refresh any open Chrome windows with Ctrl+Shift+R. (3) Disable and re-enable the extension. (4) If the issue persists, try re-loading the unpacked folder from scratch.
Results seem inconsistent — different every time I run.
+
This is expected and not a bug. AI models are non-deterministic — they generate different responses each time even for the same query. Citation counts will naturally vary between runs. For reliable trend data, run checks at the same time each month and average results across multiple runs.
