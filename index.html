FAQ

Common questions

Everything you need to know about AI Citation Tracker.

Do I need to create an account?
+

No account is required to use the web app. All your data stays locally in your browser — nothing is sent to our servers. API keys are stored securely in local storage.

How accurate is citation detection?
+

Very accurate. v2.0 introduced smart fuzzy-matching that detects your brand across multiple formats — "loom.com", "Loom", and "Loom's". This produces 3–5× more citations than simple exact-match.

Which AI platforms are supported?
+

Currently four: ChatGPT (OpenAI), Claude (Anthropic), Gemini (Google), and Perplexity. Gemini is free, the others require paid API keys from their respective platforms.

How does competitor comparison work?
+

Enter up to 5 competitor domains. The tool queries each AI platform with the same queries for every domain, then builds a side-by-side table with counts, rankings, gap scores, and recommendations.

Is my API key data private?
+

Completely. API keys are stored in your browser's local storage and never leave your device. Queries go directly from your browser to the AI provider's API — there is no server in the middle.

Can I export results?
+

Yes. From the Results tab you can copy all results to clipboard or download them as a CSV file — perfect for stakeholder reports or historical tracking spreadsheets.
